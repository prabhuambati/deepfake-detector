# 🔍 Deepfake Detection — Complete Project

A multi-branch deepfake detection system combining **CLIP visual semantics**, **temporal anomaly vectors**, and **rPPG physiological cues**.

## ⚙️ Setup on Lightning AI (T4 GPU)

```bash
# 1. Clone repo
git clone <your-repo> deepfake_detector && cd deepfake_detector

# 2. Install dependencies
pip install -r requirements.txt

# 3. Kaggle API
pip install kaggle && mkdir ~/.kaggle
cp kaggle.json ~/.kaggle/ && chmod 600 ~/.kaggle/kaggle.json

# 4. Download FaceForensics++
kaggle datasets download -d xdxd003/download-ff-mega \
  --path ./data/raw --unzip

# 5. Download CelebDF-v2
kaggle datasets download -d reubensuju/celeb-df-v2 \
  --path ./data/celebdf --unzip

# 6. Verify everything
python verify_setup.py
```

## 🚀 Execution Order

```bash
# Phase 1 — Preprocessing
python src/preprocess.py --test_run   # sanity check (10 videos/folder)
python src/preprocess.py              # full run (~30-60 min)
python src/dataset.py                 # verify batch shapes

# Phase 2 — Branch tests
python src/clip_branch.py             # expect shapes (2,512) and (2,16)
python src/rppg_branch.py             # expect shape (2,128)

# Phase 3 — Fusion & Training
python src/fusion.py                  # expect (2,2) logits
python src/train.py                   # full training (~2-4 hrs on T4)

# Phase 4 — Evaluation
python src/evaluate.py --eval_all
python src/evaluate.py --video data/raw/original/000.mp4
python src/evaluate.py --video data/raw/Deepfakes/000.mp4
```

## 🏗️ Architecture

```
Input (batch, 16, 3, 224, 224)
         │
    ┌────┴─────────────┐
    │                   │
 CLIPBranch         rPPGBranch
 (frozen ViT-B/32)  (R(2+1)D-18, layer4 trainable)
    │                   │
 mean_emb (512)     rppg_feats (128)
 anomaly_vec (16)          │
    │            × sigmoid(reliability_weight)
    └────────────────┬──────────────────────┘
                     │ concat → (656,)
                 FusionMLP
              (656→256→64→2)
                     │
               logits (2,), confidence
```

## 🔧 Common Fixes

| Error | Fix |
|-------|-----|
| CUDA OOM | Set `BATCH_SIZE = 4` in `train.py` |
| MTCNN misses all faces | Lower thresholds: `[0.5, 0.6, 0.6]` |
| AUC stuck at 0.5 | Check label distribution in CSV |
| Loss is NaN | Lower LR to `1e-5`, check for zero-division |
| `open_clip` import error | `pip install open-clip-torch --upgrade` |
| R(2+1)D slow on CPU | `torch.backends.cudnn.benchmark = True` (already set) |
